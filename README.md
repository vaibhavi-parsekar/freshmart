# freshmart
FreshMart is an online fruits and vegetables shopping platform where users can browse fresh products, add them to cart, and place orders online.

## Features

* Browse fresh fruits and vegetables
* View detailed product information
* Add products to cart
* Place orders online
* Simple and user-friendly interface
* Separate frontend and backend architecture
* Admin panel for managing the system

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript
* React

### Backend

* Java
* Spring Boot

### Database

* SQL

## Project Modules

### User Module

The user module allows customers to:

* Browse available fruits and vegetables
* View product details
* Add products to the cart
* Place orders online
* Experience a simple and responsive user interface

### Admin Module

The admin module provides full control over the system. The admin can:

* Add, update, and delete products
* Manage fruits and vegetables inventory
* View and manage customer orders
* Manage user information
* Maintain the overall system

## Project Structure

freshmart_fruits_vegetables_ordering_system
│
├── frontend (React Application)
└── backend (Spring Boot Application)

## Purpose of the Project

The purpose of this project is to develop an **online platform for ordering fresh fruits and vegetables**, making it easier for customers to shop from home while allowing administrators to efficiently manage products, users, and orders.
